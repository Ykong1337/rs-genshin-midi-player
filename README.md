# Lyred

#### Lyred is a MIDI player in various games

# Features

- Graphical user interface
- ~~Auto tuning~~
- Manual tuning
- Thread operations
- Multi-mapping
- Split track playback

Currently supported mappings: **Genshin**, **VRChat-中文吧**
